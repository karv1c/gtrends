Quick Start

Make a Get request to a following url with your keywords comma separated:

http://gtrendsapi.com/api/?keywords=

For example, to get data for keywords "Dog" and "Cat" use this request:

http://gtrendsapi.com/api/?keywords=dog,cat
Note, that you can compare as many words as you want. For more usage information visit the API page.
