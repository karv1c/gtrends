How to use this API

Make sure, that you send a GET or POST request to:

http://gtrendsapi.com/api/?

Then add some options to your request:

Time Interval

Type time= and choose the necessary time interval

http://gtrendsapi.com/api/?time=now+1-H

Here is a list of available intervals:

now+1-H1 hour time interval
now+4-H4 hours time interval
now+1-d1 day time interval
now+7-d7 days time interval
today+1-m1 month time interval
today+3-m3 months time interval
today+12-m12 months time interval
today+5-y5 year time interval
YYYY-MM-DD+YYYY-MM-DDCustom time intermal. Ex. 1989-01-05+1992-02-10
YYYY-MM-DDTHH+YYYY-MM-DDTHHCustom time intermal with hours. Ex. 1989-01-05T07+1992-02-10T08

Request is case sensetive. 1-h will not work as 1-H

Geography

Add geo= tag for choosing geography. It is worldwide by default, so don't use this option in that case.

http://gtrendsapi.com/api/?&geo=ru

Combined with time interval:

http://gtrendsapi.com/api/?time=now+1-H&geo=ru

Available geography:

ruRussian Federation

Timezone

To specifythe timezone use tag tz= with you timezone in minutes in Greenwich Mean Time. 0 is for GMT+0, 180 for GMT+3. Default timezone is GMT+0.

http://gtrendsapi.com/api/?&tz=180

Combined with time interval and geography:

http://gtrendsapi.com/api/?time=now+1-H&geo=ru&tz=180

Keywords

Place keywords comma separated after the keywords= tag without spaces. You can use as many words, as you want.

http://gtrendsapi.com/api/?&keywords=rust,blockchain

Combined with time interval, geography and timezone:

http://gtrendsapi.com/api/?time=now+1-H&geo=ru&tz=180&keywords=rust,blockchain

If you make a POST request, use neccesary options as above and send JSON keywords instead:

{"keywords": ["rust", "blockchain"]}

Response data

The response data would contain a JSON with array of points with timestamp in seconds, formatted time and array of values order of responsed list of keywords. Example below is a part of responsed data

{"data":[{"formatted_time":"Aug 14, 2022","time":1660435200,"value":[61,15]},{"formatted_time":"Aug 21, 2022","time":1661040000,"value":[57,16]},{"formatted_time":"Aug 28, 2022","time":1661644800,"value":[57,15]},{"formatted_time":"Sep 4, 2022","time":1662249600,"value":[56,16]},{"formatted_time":"Sep 11, 2022","time":1662854400,"value":[54,15]}],"titles":["Rust","Blockchain"]}
